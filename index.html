Elements

HTML uses elements to describe the structure of pages

An element is composed of a opening tag, content, and a closing tag.

Opening Tag

The character(s) defines the tags purpose.

Closing Tag

Tag Attributes

Attributes add extra information to an element. They always appear with the opening tag.

HTML, HEAD, BODY

Every web page is structured with the following elements.

HEAD

The HEAD is where the "Think'n Stuff" lives

  • Title
  • Imports (CSS & JavaScript)
  • Other Think'n Stuff (Google Analytics.....)

BODY

The BODY is for "Do'n Stuff"

  • Visible "Blocks"
  • User Interface (UI) to accept actions

HTML Summary

  • HTML creates structure for the web (think newspapers or Legos)
  • HTML is made up of elements
  • HTML elements have an opening and closing tags
  • HTML tags can include attributes
  • HTML is made up of html, head, and body elements
